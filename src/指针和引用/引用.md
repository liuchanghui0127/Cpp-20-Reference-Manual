## **4.1引用**

引用相当于变量的另一个别名，对引用的修改会改变其引用的变量的值，引用变量声明如下所示：
```
int x{3};
int& xREF{x};
```
上述代码将xREF初始化为指向x的引用，两者指向同一个值，引用必须在声明时进行初始化。

### **4.1.1引用修改**

在对引用变量进行修改时，实际修改的是其指向的变量的值，示例如下所示：
```
int x{3},y{4};
int& xREF{x};
xREF=y;
cout<<format("xREF:{},x:{}",xREF,x);
```
在将y赋值给引用变量xREF后，实际更改的是变量x，所以输出时xREF与x的值都为4。

### **4.1.2引用传递**

常规的值传递通过对值进行复制生成副本的形式进行参数传递，在面对复杂对象传递时会造成资源浪费。

而通过引用传递，可以将参数传递且不需要进行复制，如下所示：
```
void printString(const string& mystring){
  cout<<mystring<<endl;
}
int main(){
  string somestring{"Hello world"};
  printString(somestring);
}
```
注意：使用引用传递参数时修改引用则将直接修改引用参数本身，因此如果只是传递参数而并非修改参数本身，应当使用const引用传递,即采用上述引用方法。
